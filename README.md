# Loan Default Prediction System

## 📌 Overview

This project predicts whether a loan application will be approved or not using Machine Learning techniques.

## 🚀 Workflow

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Selection
* Model Training & Evaluation

## 🧠 Models Used

* Logistic Regression
* Naive Bayes
* K-Nearest Neighbors (KNN)

## 📊 Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score

## 🏆 Result

Among the models tested, Logistic Regression performed best for this dataset.

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

## ▶️ How to Run

1. Clone the repository
2. Install dependencies:
   `pip install -r requirements.txt`
3. Open and run the Jupyter Notebook

## 📁 Dataset

Loan approval dataset containing applicant details like income, credit history, etc.
